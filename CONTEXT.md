# 東葛湯めぐりマップ

東葛地域の温浴施設を、地図と一覧から探しやすくするための文脈。施設情報を静的データとして管理し、利用者とコントリビューターが同じ言葉で話せるようにする。

## Language

**温浴施設**:
このサービスで掲載する入浴・温浴系施設の総称。スーパー銭湯、日帰り温泉、サウナ併設施設などを必要に応じて含める。
_Avoid_: スーパー銭湯, 銭湯っぽい施設, スパ

**日帰り温浴施設**:
初期版で優先して掲載する温浴施設。一般利用できるスーパー銭湯、日帰り温泉、浴槽を主目的にしたサウナ併設施設を含み、宿泊者専用や会員制の浴場は原則含めない。
_Avoid_: 宿泊者専用浴場, 会員制浴場

## Example Dialogue

開発者: 新しい温浴施設を追加したいです。  
ドメイン担当: 初期データは私がMarkdown表で手動共有します。日帰り温浴施設として扱えるものを、施設名、市区町村、住所、Google Maps URL、確認日と一緒に入れてください。
